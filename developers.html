Installation Guides

Instructions for installations and builds for all componenets of Black Lager

Components of the Black Lager project

You will need the following two componenets to run the Black Lager project.

  • LoRa radio running the Black Lager firmware
  • The Black Lager Messaging client

Protobufs

Run pip install blacklager . All protocol buffer messages are defined in the central protobuf repository (link). In the Python and firmware repositories, there are Github actions defined called update Protobufs, which will pull the latest changes from the Protobuf repository and compile them.
